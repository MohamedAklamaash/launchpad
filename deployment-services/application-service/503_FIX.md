# 503 Service Unavailable Fix

## Issues Identified

1. **Health Check Failing**
   - ALB health check path: `/`
   - NGINX only responded to: `/{app-name}/*`
   - Result: All targets marked unhealthy → 503

2. **NGINX Config Shell Escaping**
   - Multi-line config passed via environment variable
   - Shell escaping issues with special characters
   - NGINX failed to start properly

## Fixes Applied

### 1. NGINX Health Check Endpoint

Added root path handler for ALB health checks:

```nginx
location = / {
    proxy_pass http://localhost:8000/;
    # Proxies health check to app
}
```

Now:
- ALB checks `/` → NGINX responds → proxies to app → healthy ✅

### 2. Base64 Encoded Config

Changed from:
```bash
echo "$NGINX_CONFIG" > /etc/nginx/nginx.conf
```

To:
```bash
echo "$NGINX_CONFIG_B64" | base64 -d > /etc/nginx/nginx.conf
```

Benefits:
- No shell escaping issues
- Handles special characters correctly
- Reliable config injection

### 3. Environment Variables

Environment variables are correctly passed:
```python
envs = application.envs or {}  # User's envs from API
envs['PORT'] = str(application.port)  # Auto-injected
```

User's envs (FINNHUB_API_KEY, etc.) are passed to the application container, not NGINX.

## Architecture

```
ALB Health Check (/)
    ↓
NGINX (port 80)
    ↓ proxies / to app
App Container (port 8000)
    ↓ responds 200
NGINX → ALB
    ↓
Target marked Healthy ✅
```

## Request Flow

```
User: http://alb/apexmarket-v1/static/app.js
    ↓
ALB → NGINX (port 80)
    ↓
NGINX strips prefix: /static/app.js
    ↓
App Container (port 8000)
    ↓ serves static file
NGINX → ALB → User ✅
```

## Common Issues

### Still 503?
- Check ECS task logs for NGINX container
- Verify app container is running
- Check app responds to `GET /` (health check)

### Envs not working?
- Verify they're in the application record
- Check ECS task definition environment variables
- Look at app container logs

### Static files 404?
- App must serve static files
- Check app's static file configuration
- Verify paths are correct
