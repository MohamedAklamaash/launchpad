## Worker Process Flow

```
┌─────────────────────────────────────────────────────────────────┐
│                    WORKER PROCESS                                │
│                                                                  │
│  1. Poll Queue                                                   │
│     ┌──────────────────────────────────────────┐               │
│     │ job = InfraQueue.dequeue_provision()     │               │
│     └──────────────────┬───────────────────────┘               │
│                        │                                         │
│  2. Authenticate       ▼                                         │
│     ┌──────────────────────────────────────────┐               │
│     │ authenticate_infrastructure(infra)       │               │
│     └──────────────────┬───────────────────────┘               │
│                        │                                         │
│  3. Generate Config    ▼                                         │
│     ┌──────────────────────────────────────────┐               │
│     │ tf_config = _generate_config()           │               │
│     │ workspace = /dev/shm/tf-{infra_id}       │               │
│     └──────────────────┬───────────────────────┘               │
│                        │                                         │
│  4. Execute Terraform  ▼                                         │
│     ┌──────────────────────────────────────────┐               │
│     │ terraform init                           │               │
│     │ terraform apply -auto-approve            │               │
│     └──────────────────┬───────────────────────┘               │
│                        │                                         │
│                        ├─── Success ───┐                        │
│                        │                │                        │
│                        │                ▼                        │
│                        │    ┌──────────────────────┐            │
│                        │    │ terraform output     │            │
│                        │    │ Save to DB           │            │
│                        │    │ status = READY       │            │
│                        │    │ Notify user          │            │
│                        │    └──────────────────────┘            │
│                        │                                         │
│                        └─── Failure ───┐                        │
│                                         │                        │
│                                         ▼                        │
│                             ┌──────────────────────┐            │
│                             │ terraform destroy    │            │
│                             │ status = FAILED      │            │
│                             │ Log error            │            │
│                             │ Notify user          │            │
│                             └──────────────────────┘            │
│                                                                  │
│  5. Cleanup                                                      │
│     ┌──────────────────────────────────────────┐               │
│     │ rm -rf /dev/shm/tf-{infra_id}            │               │
│     └──────────────────────────────────────────┘               │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## Data Flow

```
┌──────────┐
│  Client  │
└────┬─────┘
     │ POST /infrastructures
     ▼
┌─────────────────┐
│   API Server    │
│  1. Validate    │
│  2. Save to DB  │
│  3. Enqueue job │
└────┬────────────┘
     │
     ▼
┌─────────────────┐
│  Redis Queue    │
│  [job pending]  │
└────┬────────────┘
     │
     ▼
┌─────────────────┐
│     Worker      │
│  1. Dequeue     │
│  2. Provision   │
│  3. Update DB   │
└────┬────────────┘
     │
     ├─── Success ───┐
     │                │
     │                ▼
     │         ┌─────────────┐
     │         │ DB: READY   │
     │         │ Notify user │
     │         └─────────────┘
     │
     └─── Failure ───┐
                     │
                     ▼
              ┌─────────────┐
              │ Rollback    │
              │ DB: FAILED  │
              │ Notify user │
              └─────────────┘
```
